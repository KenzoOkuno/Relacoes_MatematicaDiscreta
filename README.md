# Analisador de Relações (Matemática Discreta)

Este projeto é um programa simples em Python que permite analisar **propriedades de relações** em um conjunto, um tema fundamental em Matemática Discreta.

O programa permite que o usuário:
- Defina um conjunto de elementos.
- Informe os pares ordenados que formam a relação.
- Verifique automaticamente propriedades importantes da relação.

## Funcionalidades

- **Verificação de Reflexividade**: Checa se todo elemento se relaciona consigo mesmo.
- **Verificação de Simetria**: Confere se, para todo (a, b), existe (b, a).
- **Verificação de Transitividade**: Garante que, se (a, b) e (b, c) existem, então (a, c) também existe.
- **Entrada personalizada**: O usuário pode inserir qualquer conjunto e relação.

## Como usar

1. Certifique-se de ter o Python 3 instalado.
2. Salve o arquivo como `relacoes.py`.
3. Execute o programa no terminal ou IDE:
   ```bash
   python relacoes.py
